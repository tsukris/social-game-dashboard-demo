
## KPIツリー

★KGI：売上
├─ 売上関連
│  ├─ 〇ARPU（売上 ÷ ユーザー数）
│  ├─ ★売上予測・目標差
│  │  ├─ 月末予測売上
│  │  ├─ 現時点目標売上
│  │  ├─ 月末目標売上
│  │  ├─ 目標達成率
│  │  ├─ 予測差額（月末予測売上 - 月末目標売上）
│  │  ├─ 予測更新日
│  │  └─ 予測対象期間
│  └─ 〇施策別売上
│     ├─ 施策別現時点売上
│     ├─ 施策別予測売上
│     ├─ 施策別予測差
│     ├─ 販売カテゴリ別売上構成
│     ├─ 〇商品別売上
│     └─ ★ガチャ別売上
├─ ユーザー数関連
│  ├─ 〇MAU（月間アクティブユーザー数）
│  │  └─ 〇DAU（1日のアクティブユーザー数）
│  │     ├─ ログインユーザー（ログイン勢）
│  │     │  ├─ ログインのみユーザー数（ログインはしたが、実プレイまで進んでいないユーザー数）
│  │     │  └─ デバイス別ログイン数（iOS / Android / PC など）
│  │     └─ ユーザー区分粒度（詳細は末尾の「分析軸」を参照）
│  ├─ ★rMAU（ログインのみを除いた月次実プレイユーザー数）
│  │  └─ ★rDAU（ログインのみを除いた日次実プレイユーザー数）
│  ├─ 〇ログインユーザー率（DAU-rDAU/DAU）
│  ├─ 〇新規定着ユーザー数（新規ユーザー数×新規定着率）
│  │  ├─ ★新規ユーザー数
│  │  │  ├─ 〇広告由来流入数
│  │  │  │  ├─ 広告表示回数
│  │  │  │  ├─ 広告クリック率（CTR：広告が表示された回数のうち、クリックされた割合）
│  │  │  │  ├─ インストール転換率（CVR：広告クリック後に実際にインストールされた割合）
│  │  │  │  └─ 広告獲得単価（CPI：1インストールを獲得するためにかかった広告費）
│  │  │  ├─ 〇自然流入数（広告以外。口コミ、SNS、ストア検索などから入ってきた数）
│  │  │  │  ├─ SNS由来流入数
│  │  │  │  ├─ ストア検索由来流入数
│  │  │  │  ├─ コラボ由来流入数
│  │  │  │  ├─ 口コミ / 外部コミュニティ由来流入数
│  │  │  │  └─ 復帰キャンペーン由来流入数
│  │  │  ├─ 新規プレイヤー1人あたり売上（広告効果の価値換算に使う。売上 ÷ 新規ユーザー数）
│  │  │  └─ 年始比新規ユーザー数（現在のrDAUに含まれるが、年始時点のrDAUには含まれていなかったユーザー数）
│  │  └─ 〇新規定着率
│  │     ├─ 〇チュートリアル突破率
│  │     └─ ★D1 / D7 / D30継続率
│  │        ├─ D1継続ユーザー数（D1_RETAINED_USERS）
│  │        ├─ D1コホート人数（D1_COHORT_SIZE）
│  │        ├─ D1継続率（D1_RETENTION_RATE）
│  │        ├─ D7継続ユーザー数（D7_RETAINED_USERS）
│  │        ├─ D7コホート人数（D7_COHORT_SIZE）
│  │        ├─ D7継続率（D7_RETENTION_RATE）
│  │        ├─ D30継続ユーザー数（D30_RETAINED_USERS）
│  │        ├─ D30コホート人数（D30_COHORT_SIZE）
│  │        └─ D30継続率（D30_RETENTION_RATE）
│  ├─ 〇休眠復帰ユーザー数（一定期間非アクティブだったが、再度アクティブ化したユーザー数）
│  └─ ★既存顧客離脱数（当日のユーザー数 - 新規ユーザー数 - 前日のユーザー数）
│     ├─ 潜在離脱顧客数（離脱する可能性のあるプレイヤーの人数）
│     ├─ 年始比既存顧客離脱数（年始時点のrDAUに含まれていたが、現在のrDAUには含まれていないユーザー数）
│     ├─ ユーザーの熱量関連
│     │  ├─ 〇参加率（クエスト / イベントなど主要コンテンツへの参加率）
│     │  ├─ 〇クリア率
│     │  │  └─ クエスト別難易度（特定基準のエリートプレイヤーのうち、当該クエストをクリアできた割合）
│     │  ├─ 〇エピソード閲覧率（個別エピ / 親密度エピソードの閲覧率）
│     │  └─ 〇SNS動向
│     │     ├─ SNS投稿数
│     │     ├─ 公式SNSポスト反応数（自社ポストへのいいね、RP、返信など）
│     │     ├─ 公式SNSポストエンゲージメント率
│     │     └─ SNSネガポジ分析
│     ├─ ユーザー満足度関連
│     │  ├─ 〇キャラ満足度
│     │  │  └─ 個別キャラ満足度
│     │  ├─ 〇親密度エピソード満足度
│     │  │  └─ 個別キャラ満足度
│     │  ├─ 〇シナリオ満足度
│     │  │  └─ 個別シナリオ満足度
│     │  ├─ 〇ゲーム満足度
│     │  │  └─ 個別コンテンツ満足度（対人 / レイド / ランキング戦 など）
│     │  └─ 〇課金商品満足度
│     │     └─ 個別商品満足度
│     └─ 技術品質関連
│        ├─ ★クラッシュ率
│        └─ ★平均読み込み時間
└─ 課金関連
   ├─ ★課金率（DPU/rDAU）（実プレイユーザーのうち課金した割合）
   │  ├─ 〇DPU（課金UU）
   │  └─ 課金者区分（人数・売上・比率。上記「課金額」区分を参照）
   │     ├─ 廃課金
   │     ├─ 重課金
   │     ├─ 軽課金
   │     ├─ 微課金
   │     └─ 無課金
   ├─ ガチャ導線
   │  └─ 〇ガチャページクリック率
   │     └─ 〇ガチャ購入率
   ├─ 商品導線
   │  └─ 〇商品ページクリック率
   │     └─ 〇商品購入率
   ├─ ★ARPPU（課金者1人あたり平均売上）
   │  ├─ 1回あたり購入単価
   │  ├─ 課金頻度
   │  ├─ ユーザーの熱量関連（再掲：課金者単価に効く活動量）
   │  └─ ユーザー満足度関連（再掲：課金者単価に効く満足/不満）
   ├─ 課金ポテンシャル
   │  └─ 予想廃向け最大課金金額（月次：理論上、または設計上想定される1人あたり月間課金上限）
   │     ├─ 有償販売物総量（月内に販売される有償商品・パック・月額商品などの総額）
   │     └─ 新規ガチャ必要回転数総量（月内に新規追加ガチャを狙うために必要な総回転数）
   └─ 石関連
      ├─ 石配布数
      ├─ 石収支（石配布数 - 石消費数）
      ├─ 石消費数（有償：GACHA_PAID_SPEND）
      ├─ 石消費数（無償：GACHA_FREE_SPEND）
      ├─ ★石消費数（合計：GACHA_TOTAL_SPEND）
      ├─ ガチャ実行回数
      ├─ 天井到達率
      └─ 〇石保有数（rDAU_AVG_STONE_HOLDINGS）

## 分析軸（KPIではなく、各KPIを切り分けるための軸）

- 時間（年 / 月 / 週 / 日 / 曜日 / 時間帯 / 月初・月中・月末 / イベント前・中・後 / 周年期 / コラボ期）
- プラットフォーム（PCブラウザ / PCアプリ / スマホブラウザ / iOS / Google Play / プラットフォームD）
- ユーザー分類（課金状態 / プレイ継続期間 / プレイ頻度 / プレイ深度 / 攻略進捗 / キャラ資産 / ガチャ行動 / 施策反応 / 独自ユーザー分類 / リスク分類）
- 販売物（ガチャ / 商品 / 月額パス / 石パック / 育成素材 / 衣装 / 復刻 / 限定）
- 時間比較（前日比 / 前週比 / 前月比 / 前年比 / 同日平均 / 直近12か月同日平均 / 目標比 / 予測比）
